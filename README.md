# mazeGame

A basic maze terminal game

### Prerequisites

* Install .NET Core SDK https://www.microsoft.com/net/core

### Installing

* `dotnet restore`
* `dotnet run`

## Built With

* [dotnetcore](https://www.microsoft.com/net/core)
